# Aquila

Manage Docs Seamlessly. Aquila Helps you to manage testable code snippets in your docs.

## Usage

See Usage Instructions.

Building from Source :

```bash
  git clone git@github.com:amanv8060/aquila.git
  cd aquila
  go build
```
